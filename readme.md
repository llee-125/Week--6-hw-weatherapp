## Weather App
Create a weather app using OpenWeather API.

## Motivation
Individuals should be able to search for a city's current weather and receive:
* City name
* City date
* Weather icon
* Temperature
* Humidity
* Wind Speed
* UV Index
* 5 day forcast
* Searched cities should be rendered and stored on local storage.
